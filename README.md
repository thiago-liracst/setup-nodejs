# Setup NodeJS
Setup para aplicações NodeJS utilizando TypeScrip
# Start
Instalação das dependências </br>
`yarn`</br>
Build Dev usando tsnode-typescrip</br>
`yarn dev` </br>
Build PRD usando Babel </br>
`yarn build` </br>
Rodar testes </br>
`yarn test` </br>
Iniciar aplicação PRD </br>
`yarn start` </br>
